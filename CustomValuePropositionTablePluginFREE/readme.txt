== Description ==
Create a beautiful and responsive FAQ section using Blupry's lightweight accordion plugin. 
Perfect for common questions, product support, services, and more.

== Features ==
- Custom FAQ input
- Accordion-style toggles
- Choose animation style
- Primary color customization
- Modern design with optional icons
